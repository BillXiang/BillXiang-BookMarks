Saniffer - SSD, Server, Networking, Storage & Application Analysis, Test and Monitoring Solution Expert

    Saniffer is a turn-key solution provider for test tools focusing on SSD, server, networking, storage and application performance analysis, test and monitoring. 

 

    Saniffer's products meet test requirements of most customers from product design and test in server, network, storage and mobile fields, to deployment and operation of applications in data centers . Our products cover solutions like Storage Bus protocol analyzer, jammer, host emulator, target emulator, exerciser, BERT, load tester and the overall IT application performance monitoring, etc.  Saniffer also released the most recognized SAS/SATA/NVMe SSD test tools white paper starting from early 2017 - SSD Test Tools White Paper and Quick Start Guide. 

 

    Saniffer is the distributor of the following test solutions on SAS/SATA, PCIe/NVMe SSD, SD/SDIO/eMMC, UFS, DDR/LP-DDR technologies, and also distributor for Fibre Channel, iSCSI, FCoE, NVMe over Fabric (NVMoF), SMB/CIFS/NFS test solutions.  

  • SerialTek                       - PCIe/NVMe Gen4/5 and SAS/SATA protocol analyzer
  • SanBlaze                     - PCIe/NVMe Gen 4/5and SAS/SATA SSD Test Systems
  • Quarch                          - SAS/SATA, USB, PCIe, SFF 8639 SSD Hotplug Automation Test
  • SerialCables               - PCIe/NVMe Gen 4/5 and SAS/SATA adapters and HBAs
  • NplusT                           - Nand Flash Characterization Analyzer and Tester
  • Nexus                            - DDR 3/4/5 and LPDDR 3/4/5 protocol analyzer and fixtures
  • Kazan Networks        - NVMoF (NVMe over Fabric 100GE) Target Solution
  • DinoStor                       - NDMP SDK and Compliance Test Suite, Data Migrator
  • CN-100 Tester             - TTE tester for TTE Switch performance and latency Test
  • SCSI Test Box              - Peripheral Test for FC, SAS/SATA, PCIe, FCoE/iSCSI Device
  • Load Dynamix            - FC, iSCSI, NAS(SMB/CIFS/NFS), HTTP(S), Amazon S3 Storage Test
  • Echola System           - Fibre-cut Simulation Switch, Fibre Matrix Switch
  • GigaNet Systems      - FC SAN, TCP/IP Network Impairment & Latency Emulation
  • LTTS                                - LTTS IP for  CSI/DSI/I3C/SoundWire/UFS/USB/SATA

    Saniffer distributes RapidIO protocol analyzer which is widely used in embedded system design fields, including storage, networking, aerospace, telecom, post silicon verification, etc. 

  • RapidIO Analyzer                 - RapidIO Gen 2 x4 Protocol Analyzer and Generator 

    The BTS (Batch Test System) focuses on server, storage and laptop automation test in System Integration Test department, which could could expedite test speed greatly and save as much as 75% of test time. BTS covers automated OS installation, driver installation and component performance and function test like CPU, DDR 4, PCIe, Drive, NIC, Display, etc. 

  • Batch Automation Test      - Batch Test System for Server System Integration Test     

    Saniffer is the distributor of Uila, which is famous for its Application-Aware Virtual Infrastructure Performance Management software. 

  • Uila                                  - Application Aware Infrastructure Performance Management
