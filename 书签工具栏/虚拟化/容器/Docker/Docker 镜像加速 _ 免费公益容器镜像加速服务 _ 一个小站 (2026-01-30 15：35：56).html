快速使用

docker pull docker-pull.ygxz.in/library/nginx:latest

💡 请手动复制上述命令到终端执行

温馨提示： 本服务仅供学习研究使用，请合理使用，避免滥用。

全局配置方法

  1. 编辑或创建 Docker 配置文件：
    sudo nano /etc/docker/daemon.json
  2. 添加以下配置：
    { "registry-mirrors": ["https://docker-pull.ygxz.in"] }
  3. 重启 Docker 服务：
    sudo systemctl daemon-reload sudo systemctl restart docker

💡 请手动复制上述命令到终端执行
