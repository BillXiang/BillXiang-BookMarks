Wormhole™

The Wormhole™ n150 and n300 PCIe boards are flexible, scalable processors built with Tensix Cores™. Each includes a compute unit, network-on-chip, local cache and “baby RISC-V” cores, coalescing in powerful data movement through the chip. Wormhole™ offers superior performance for cost compared to traditional GPUs, and broad data precision format support.

Wormhole™ n150d

$1,099

Wormhole™ n150d features Tenstorrent’s flexible, scalable Wormhole™ Tensix Processor operating at up to 160W, offering superior performance for cost compared to traditional GPUs and broad data precision format support. The processor can network into a multichip mesh for workstations and servers (such as Galaxy) and is supported by Tenstorrent’s fully open source software stack, granting developers full access to the metal. n150d includes a 2.5-slot active cooling solution.
