词源[编辑]

‫٠ (0)‬阿拉伯数字尼泊尔形式的“西部阿拉伯”变体（对照天城文 ०），在欧洲与拉丁文字一起使用。

符号[编辑]

0

  1. (阿拉伯数字) 基数零
  2. 在十进制或其他进制的计数系统中的数字零
    15 × 134 = 2010
    0000 00012 (二进制)
    0x1000E001 (十六进制)
  3. (数学) 关于加法的单位元
    ∃ 0 : ∀ a ∈ A : 0 + a = a + 0 = a {\displaystyle \exists \,\mathbf {0} :\forall a\in A:\mathbf {0} +a=a+\mathbf {0} =a}
  4. (数学，计算机科学) 对应假的布尔值或真值
    f : Bk → B : B = {0, 1}
    A&B&C=0 → A=0 || B=0 || C=0
  5. (计算机) 表示后面的数字是八进制的
  6. (语音学，通常上标) 比1更极端的音调（高或低取决于当地的习惯，例如：0在查蒂诺是最高音。）
  7. (语音学，汉字文化圈，通常上标) 缺乏内在的音调；音高取决于周围音节的音调

发音[编辑]

  • (号码, 数字, “zero”)：
    • (澳大利亚，美国) 英语发音：zē'rō, 国际音标(帮助)：/ˈziːɹəʊ/
    • (标准英音) 国际音标(帮助)：/ˈzɪəɹəʊ/
    • 音频 (美国)：时长：1秒。0:01(档案)
    • 韵部：-ɪəɹəʊ
  • (数字, “nought”)：
    • (标准英音) 国际音标(帮助)：/nɔːt/
    • 韵部：-ɔːt
  • (数字, “oh”)：
    • (标准英音) 国际音标(帮助)：/əʊ/
    • 韵部：-əʊ
    为了节省时间或保持节奏，作为一个数字的发音经常被缩短为字母O的发音。在小时之后的分钟的十位和10到1900之间的数字的缩短发音中，包括年，以及它们的截断两位数形式，都需要这种变化，其中零是一个前导数字。在说电话号码、小数点后的小数、小于1的小数分数中的前导零，或许多其他词语如007（double o seven）时，它也很常见。

名词[编辑]

0 (复数 0s)

  1. 数字零
  2. 原点、起点或固定的参考点，特别是对于一个测量来说。
    The 0 for the Celsius scale is 273.15 Kelvin.
  3. (电气工程) 关闭或低位状态。
    If the NOT gate input is 1, the output is 0.
