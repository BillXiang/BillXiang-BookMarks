go http 返回json

在Go语言中，你可以使用net/http标准库来创建HTTP服务器，并使用encoding/json标准库来处理JSON数据。以下是一个简单的例子，展示了如何设置一个HTTP服务器，该服务器接收JSON请求并返回JSON响应：

package main
 
import (
    "encoding/json"
    "net/http"
)
 
// 定义你的数据结构
type ResponseData struct {
    Status  string `json:"status"`
    Message string `json:"message"`
}
 
func main() {
    http.HandleFunc("/json", jsonHandler)
    http.ListenAndServe(":8080", nil)
}
 
func jsonHandler(w http.ResponseWriter, r *http.Request) {
    var response ResponseData
 
    // 解析请求体中的JSON数据
    err := json.NewDecoder(r.Body).Decode(&response)
    if err != nil {
        http.Error(w, err.Error(), http.StatusBadRequest)
        return
    }
 
    // 在这里可以处理response中的数据...
 
    // 设置响应头来指示返回的内容是JSON
    w.Header().Set("Content-Type", "application/json")
 
    // 设置状态码
    w.WriteHeader(http.StatusOK)
 
    // 返回JSON格式的响应
    json.NewEncoder(w).Encode(map[string]string{
        "status":  "success",
        "message": "JSON handled successfully",
    })
}

在这个例子中，我们定义了一个jsonHandler函数来处理路径为/json的HTTP请求。该函数首先解析请求体中的JSON数据，然后设置响应头，并返回一个JSON格式的响应。这是一个非常基础的示例，实际应用中你可能需要根据需求来扩展处理逻辑和响应数据。
